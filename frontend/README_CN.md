# Law Compare 前端

基于 Next.js 构建的现代化法律文档比对可视化界面。

[English Version](./README.md)

---

## ✨ 特性

### 1. 多样化可视化模式
- **结构化比对视图**：功能最强大的视图。展示完整的文档层级，并高亮章、节、条的新增、移动或变更。侧边栏导航支持快速跳转。
- **左右对照视图**：针对阅读体验优化。同步滚动与对齐的文本块让措辞变动一目了然。
- **Git 经典视图**：经典的红绿对比颜色，适合习惯传统 Diff 工具的用户。

### 2. 交互与用户体验
- **动态状态指示**：悬浮标签展示每个法条的相似度百分比。
- **实体高亮**：对重要实体（日期、金额等）进行颜色编码，辅助快速识别实质性修改。
- **自适应设计**：基于 Tailwind CSS 构建，支持深色/浅色模式，采用极具质感的磨砂玻璃设计风格。
- **多语言**：完整支持中英文界面切换。

## 🛠️ 技术栈
- **框架**：Next.js 15+ (App Router)
- **样式**：Tailwind CSS 4
- **动画**：Framer Motion
- **图标**：Lucide React
- **逻辑**：针对大型树状结构进行优化的 React Hooks。
